Guide

How to Buy

(click to show/hide)
  1. You must be a registered user to be able to bid on auctions. if you are not registered yet, please register here.
  2. Once you have logged in, go to the auctions tab and search for the listing you are interested in.
  3. Click the view button to see all the details regarding this listing including the ability to bid. You can also find all the bidding history and the expiration date of the auction here.
  4. Place your bid, and now you can track this listing from your profile page to see all your bidding history.
  5. If you have the highest bid by the auction's expiration date, you are now the winner.

How to Sell

(click to show/hide)
  1. You must be a registered user to be able to create an auction listing. if you are not registered yet, please register here.
  2. Once you have logged in, go to the profile tab, and expand the “My auctions” section.
  3. Click the “create new auction+” button, this will take you to the listing’s creation page. Here you need to enter all the required information and click “create”.
  4. You can monitor all the bids by accessing your listings in the profile tab over time.
